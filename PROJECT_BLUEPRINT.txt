## 2. DATABASE CONFIGURATION
### hibernate.cfg.xml
Configures the connection to MySQL and enables automatic table creation (`update`).

```xml
<hibernate-configuration>
    <session-factory>
        <property name="hibernate.connection.driver_class">com.mysql.cj.jdbc.Driver</property>
        <property name="hibernate.connection.url">jdbc:mysql://localhost:3306/PCM?useSSL=false</property>
        <property name="hibernate.connection.username">root</property>
        <property name="hibernate.connection.password">YOUR_PASSWORD</property>
        <property name="hibernate.dialect">org.hibernate.dialect.MySQLDialect</property>
        <property name="hibernate.hbm2ddl.auto">update</property>
        
        <!-- Entity Mappings -->
        <mapping class="com.pcms.model.User"/>
        <mapping class="com.pcms.model.PC"/>
        <mapping class="com.pcms.model.Request"/>
    </session-factory>
</hibernate-configuration>
```

## 3. MVC PACKAGES & CLASSES

### A. MODEL (com.pcms.model)
Entities representing database tables.

**User.java**
```java
@Entity @Table(name = "Users")
public class User {
    @Id @GeneratedValue(strategy = GenerationType.IDENTITY)
    private int id;
    private String firstName, lastName, email, password;
    @Enumerated(EnumType.STRING)
    private Role role; // ADMIN, TECHNICIAN
}
```

### B. DAO (com.pcms.dao)
Handles database operations using Hibernate.

**UserDAO.java**
```java
public class UserDAO {
    public User login(String email, String pwd) {
        try (Session s = HibernateUtil.getSessionFactory().openSession()) {
            return s.createQuery("FROM User WHERE email=:e AND password=:p", User.class)
                    .setParameter("e", email).setParameter("p", pwd).uniqueResult();
        }
    }
}
```

### C. SERVLET / CONTROLLER (com.pcms.servlet)
Handles web requests and navigation.

**LoginServlet.java**
```java
@WebServlet("/login")
public class LoginServlet extends HttpServlet {
    protected void doPost(HttpServletRequest req, HttpServletResponse resp) {
        // 1. Get parameters
        // 2. Hash password with MD5
        // 3. Call UserDAO.login()
        // 4. Set Session and Redirect
    }
}
```

### D. UTILS (com.pcms.util)
Helpers for Hibernate sessions and Security.

**HibernateUtil.java**: Manages the `SessionFactory`.
**PasswordUtil.java**: Contains `md5(String input)` for hashing.

## 5. HOW TO RUN THE APPLICATION

1.  **Environment Setup**: Install JDK 11 and MySQL 8.
2.  **Database**: Create the `PCM` database.
3.  **Config**: Update `hibernate.cfg.xml` with your MySQL username and password.
4.  **Build**: Run `mvn clean package` in the project root.
5.  **Deploy**: 
    - Copy the generated `target/PCMS.war` to the `webapps/` folder of **Apache Tomcat 10**.
    - OR use the Maven Tomcat plugin: `mvn tomcat7:run`.
6.  **Access**: Open `http://localhost:8080/PCMS/` in your browser.
